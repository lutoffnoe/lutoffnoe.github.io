Projets

Chatbot

Dans le cadre de ce projet, il fallait devélopper un Chatbot qui répondait à des questions de culture générale en utilisant une hash map. Puis il était demandé de comparer la compléxiter algoritmique de notre approche à celle de l’approche dite naïve.

Installation d’un poste pour le développement

Dans le cadre de ce projet j’ai dû installer un système d’exploitation Debian 13 ainsi qu’un environnement de développement informatique (IDE, JDK, etc) sur une machine virtuelle. Il fallait aussi faire une carte mentale afin d’expliquer comment faire cette installation.

Base de données : Le Tourmentin

Ce projet consistait à la création d’un base de données ainsi qu’à son utilisation. On a donc créer un ebase de données, on l’a alimenté, on a tester des recherches et des modification sur certaines données.

Site Web complémentaire, Econocom

Le projet était la création d'un site web complémentaire pour l’entreprise Econocom, site web déstiné aux élèves de troisième à la recherche de stage. Il y a d’abord eu une étude de l’entreprise Econocom, nous avons donc dû récupérer des informations sur Econocom ensuite reformuler ces informations pour des élèves de troisième. Après un de nos « clients » nous a énoncé ses attentes, nous avons fait une maquette en fil de fer puis, avec l’accord du client nous avons développer le site web.
